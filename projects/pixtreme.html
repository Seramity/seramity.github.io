Pixtreme- PHP

A free image hosting platform

Pixtreme was a social network built around image based content.

Pixtreme gave the user the ability to post content, follow other users, and have interaction with others and their content (favorite content and comments).

Mission

The goal was to create a beautiful platform for people to host things from art to plain images on the internet

Technologies

Pixtreme was created using PHP, no frameworks. Data and all that was managed with MySQL (phpMyAdmin).

Pixtreme was created based on the authorization framework I created for Kudobit.
